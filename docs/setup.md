## Setup & Development

### Prerequisites
- Node.js 20+
- Rust toolchain (`rustup`)
- Tauri CLI (`npm i -D @tauri-apps/cli` or `cargo install tauri-cli`)

### Install
```bash
npm install
```

### Run Desktop (Dev)
```bash
npx tauri dev
```

### Build Desktop (Prod)
```bash
npx tauri build
```

### Web-only Dev (for UI work)
```bash
npm run dev
```
